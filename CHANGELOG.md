## [v0.2.0] - 2026-07-20

### Highlights

- `devfolio devx dashboard`: build a DeveloperPeriodReport from the local OmniDevX event store (Claude Code, Codex CLI, git, GitHub) and export it as a dashforge dashboard
- MkDocs documentation site, matching the rest of the OmniDevX ecosystem
- Contributor profile: local git clone detection with automatic API fallback, multi-stage progress reporting, and a dashforge dashboard export flag

- `devfolio devx dashboard` command: reads the local OmniDevX store, builds a `DeveloperPeriodReport`, exports a portable dashforge dashboard (8 metric tiles, daily activity/cost charts, source-coverage table) ([`13ec741`](https://github.com/plexusone/devfolio/commit/13ec741))
- `output/devxdashboard` package: projects a `DeveloperPeriodReport` into `dashforge`'s `dashboardir.Dashboard` type directly (not a local reimplementation), so chart widgets render correctly in dashforge's current viewer ([`7aae21c`](https://github.com/plexusone/devfolio/commit/7aae21c))
