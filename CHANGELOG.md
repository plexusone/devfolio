- AI collaboration tracking to measure how AI-native developers are

- `contributor profile` command to generate comprehensive contributor profiles from GitHub activity ([`bb0c219`](https://github.com/plexusone/devfolio/commit/bb0c219))
- Profile types including commits, PRs, issues, reviews, and repository breakdown ([`ba04549`](https://github.com/plexusone/devfolio/commit/ba04549))
- GitHub API client for fetching user contributions and activity heatmap data ([`75da28a`](https://github.com/plexusone/devfolio/commit/75da28a))
